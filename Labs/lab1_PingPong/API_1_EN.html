Windows API - part 1

Pong

  • Example app
  • Task description:
    • application consists of three windows: main window, ball and paddle
    • moving paddle with mouse pointer
    • ball bounces off the walls of main window and paddle
    • when the ball passes the paddle, the game is over - paddle and ball stop moving
  • Functionality:
    • Main window:
      • yellow, transparency 20%
      • size: 200x300, prevent a size changing, but allow to change the window position
      • at the start main window appears in the middle of the screen
    • Ball:
      • red circle
      • changes the position of every 50ms
    • Paddle:
      • paddle in the color of the active window's title bar
      • mouse controls the middle point of paddle
      • paddle moves at the bottom edge of the main window
  • Additional requirements:
    • on the taskbar is only one window
    • application window always on top of other windows
    • combination Alt + F4 closes the application
    • show system menu when you click Alt + Spacebar, or right-click on a window in the taskbar
  • Tips:
    • SetLayeredWindowAttributes(), WS_EX_LAYERED, WS_MAXIMIZEBOX
    • SetWindowRgn(), GetWindowRect(), MoveWindow()
    • an example of creating and settings window region 
      HRGN region=CreateEllipticRgn(0,0,10,10);
SetWindowRgn(hWnd,region,true);
    • SetTimer(), WM_TIMER
    • GetCursorPos(), WM_MOUSEMOVE
    • ClientToScreen(), ScreenToClient()
    • InvalidateRect()
  • Note: In all doubtful and untold aspects application should behave like example app.
  • Scoring:
    • proper layout of main window (color, size, position) - 3 points
    • proper layout of ball (color, size, shape) - 2 points
    • proper layout of paddle - 1 point
    • ball moves - 3 points
    • paddle moves - 2 points
    • paddle bounce the ball - 1 point
