Company Data Editor

Your task is to create simple company editor, that allows administrators to change some of the basic information. It's laboratory task only (12pts). If any doubts according to task, please follow the sample application or ask your teacher.

Laboratory Part

  • Sample Application
  • Layout:
    • The application startup location is set to screen center.
    • Application's size is set and fixed at 800x450.
    • Application is divided into two areas: Left one containing list of companies and width 200px, and right one, that fills all remaining area, containing some specific info about selected company.
      • Companies list - Contains company's name.
      • Branches list - Contains branch's name. It's aligned to center and stacked up together.
      • Company Name and Phrase - non-editable text.
      • Branch Name - non-editable text in format '{Branch Name} {New Line} a {Company Name} Company'.
      • Branch Address - non-editable text.
      • Website - non-editable text.
      • Branch Contact E-mail - Editable text with e-mail address validation (simple - '.' and '@').
      • Branch Contact Phone - Editable text with phone number validation (simple - only digits).
      • Company description - Contains validation and characters counter.
    • All text in application should be italic.
  • Input Data:
    • Data is given as an .xml file.
  • Binding:
    • Use Binding for all elements and UI changes.
    • No code-behind in C# is allowed. Except that which loads data.
    • You can only use Converters and Validators.
  • Hints:
    • ListView, StackPanel, TextBlock, TextBox, IValueConverter, ValidationRule, MultiBinding, DataContext with ElementName, RelativeSource, StaticResource, ControlTemplate, DataTrigger
  • Points:
    • Input Data:
      • Deserializing data from XML - 1p
    • Layout:
      • Left area - companies list - 1p
      • Right area - branches list - 1p
    • Binding:
      • Companies list - 1p
      • Branches list - 1p
      • Branch Address - 1p
      • Branch Name - Multibinding - 2p
      • Company Description (Validation) with Characters Counter - 1p
      • Company Name (Readonly) and Phrase (Readonly) and Website - 1p
    • Validation:
      • Branch Contact - Email Validation - 1p
      • Branch Contact - Phone Validation - 1p
